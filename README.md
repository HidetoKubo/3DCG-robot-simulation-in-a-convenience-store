# 3DCG_robot_simulation
ロボットがコンビニに来店した場合の3DCGシュミレーション実験 (Unity)
ゼミで使用した実験サイトです.

「ロボットがお酒を購入する意思を示した際の，参加者の反応」を調査するために実験を行いました．

静的WEBサイトホスティングサービスNetlifyを用いて，公開しております．
